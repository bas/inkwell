# Labels

Labels are coloured tags you can attach to notes. They make it easy to group and filter notes by topic, project, or status.

## Applying labels to a note

1. Open a note.
2. Click the **Labels** button (tag icon) in the top-right of the editor header.
3. A picker drops down showing all existing labels. Click a label to toggle it on or off.
4. To create a new label and apply it at the same time, type its name in the search field inside the picker and press **↵** or click **Create "name"**.

Applied labels appear as coloured chips beneath the note title.

## Removing a label from a note

Open the label picker (see above) and click the label again to deselect it.

## Managing labels

The **Label Manager** lets you rename labels, change their colour, and delete them.

1. In the sidebar, click the **⚙︎** (gear) icon to the right of the label filter drop-down.
2. The Label Manager dialog opens, showing all existing labels.

### Renaming a label

Click the name field of a label row and type the new name, then click **Save**.

### Changing a label's colour

Click the colour chip on the right of the label row to open a colour menu. Available colours are:

`default` · `primary` · `secondary` · `accent` · `success` · `attention` · `severe` · `danger`

Select a colour to apply it. The change is reflected immediately on all notes that carry the label.

### Deleting a label

Click the **🗑** (trash) icon on the label row and confirm. The label is removed from all notes.

### Creating a label in the Manager

Click **+ New label** at the bottom of the dialog, enter a name, choose a colour, and click **Save**.

## Label colours reference

| Colour token | Appearance |
|---|---|
| `default` | Grey |
| `primary` | Blue |
| `secondary` | Purple |
| `accent` | Teal |
| `success` | Green |
| `attention` | Yellow |
| `severe` | Orange |
| `danger` | Red |

Colours adapt automatically to light and dark mode.
